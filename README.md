# 🐍 Python Object-Oriented Programming — Complete Course

A comprehensive series of **9 Jupyter notebooks** designed to take you from OOP beginner to advanced practitioner.

## 📚 Notebooks

| # | Notebook | Topics |
|---|----------|--------|
| 01 | [Introduction to OOP](./01_Introduction_to_OOP.ipynb) | What is OOP, classes, objects, attributes, methods, `self` |
| 02 | [Constructors & `__init__`](./02_Constructors_and_Init.ipynb) | Initialization, type hints, defaults, assertions, `__repr__` |
| 03 | [Class & Instance Attributes](./03_Class_and_Instance_Attributes.ipynb) | Shared vs unique data, `__dict__`, attribute lookup chain |
| 04 | [Class & Static Methods](./04_Class_and_Static_Methods.ipynb) | `@classmethod`, `@staticmethod`, factory methods, alternative constructors |
| 05 | [Inheritance](./05_Inheritance.ipynb) | `super()`, method overriding, MRO, multiple inheritance |
| 06 | [Encapsulation](./06_Encapsulation_Getters_Setters.ipynb) | Properties, getters, setters, `_protected`, `__private`, name mangling |
| 07 | [Four Pillars of OOP](./07_Four_Pillars_of_OOP.ipynb) | Abstraction (ABCs), polymorphism, duck typing, operator overloading |
| 08 | [Magic Methods](./08_Magic_Methods.ipynb) | Dunder methods deep dive, containers, callable, context managers |
| 09 | [Advanced Patterns](./09_Advanced_OOP_Patterns.ipynb) | Composition, mixins, dataclasses, design patterns (Singleton, Observer, Strategy, Factory) |

## 🎯 Each Notebook Includes

- 📖 **Detailed markdown explanations** with real-world analogies
- 💻 **Runnable code examples** with inline comments
- 📊 **Reference tables** for quick lookup
- 🏋️ **Practice exercises** with solutions
- 📌 **Key takeaways** summary

## 🚀 Getting Started

1. Make sure you have Python 3.7+ and Jupyter installed
2. Open any notebook in VS Code or JupyterLab
3. Run cells sequentially — each notebook builds on the previous one
4. Try the exercises before checking the solutions!

## 📋 Prerequisites

- Basic Python knowledge (variables, functions, loops, lists, dictionaries)
- No prior OOP experience needed — we start from scratch!
